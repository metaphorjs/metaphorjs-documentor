Function called when flag is being added to an item. file.ext.item.itemType.*flagName.add. For example: a hook file.js.item.function.param.add would be called when processing param flag and adding it to already identified function.
This hook is also called with flagName equal to item type: file.js.item.function.function.add. In its generic version this hook sets item name from the content of this flag.
